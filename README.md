# iiking-tag-bot
# Telegram Tag Bot 🤖

This is a Telegram group bot that automatically saves active users and allows tagging them using /tagall command.

---

## 🚀 Features
- Auto-saves active group members
- /tagall command to mention users
- Works only in groups/supergroups
- Simple Python-based bot

---

## 📌 Commands
- /start → Start the bot
- /tagall → Tag active group members

---

## ⚙️ Setup
1. Add bot to your Telegram group
2. Make it admin
3. Run the bot using Python
4. Set your BOT_TOKEN in main.py

---

## ⚠️ Note
- Bot only saves users who send messages in the group
- It does NOT fetch full group member list (Telegram limitation)
- Data resets on restart unless database is used

---

## 👨‍💻 Built with
- Python
- python-telegram-bot library
